# Bond Portfolio Analyzer

A single-file web app for analyzing corporate and government bond portfolios.

## Features

- CSV Upload with drag & drop
- Newton-Raphson YTM calculation
- Owned vs Available bond comparison
- Similarity matching (maturity, yield, coupon, type)
- Side-by-side comparison panel
- Sort, search, filter
- Manual bond entry
- CSV export

## Usage

1. Open bond-analyzer.html in any modern browser
2. Upload CSV or click Load sample data
3. Toggle Owned/Available filters
4. Click similar bond tags to compare
